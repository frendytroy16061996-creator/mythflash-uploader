Privacy Policy

MythFlash Uploader — last updated 12 July 2026

The short version

MythFlash Uploader runs entirely on one person's computer. It has no server, no database, and no analytics. It collects no personal information from anyone, because it has no users other than its own operator.

What is stored, and where

Data Where it lives Why
TikTok access & refresh token A local file (tokens.json) on the operator's computer So the tool can upload drafts without logging in again each time
Video files and their captions Local folders on the operator's computer They are the content being uploaded
A list of which videos were already uploaded A local file (uploaded_tiktok.json) To avoid uploading the same video twice

What is sent to TikTok

Only the video file itself, plus the access token that proves the upload is authorized. This goes directly from the operator's computer to TikTok's official API endpoints over HTTPS.

What is never done

  • No data is sold, shared, or sent to any third party.
  • No analytics, tracking, telemetry, or advertising of any kind.
  • No information is read or collected about other TikTok users.
  • No personal data of any other person is processed at all.

Deleting your data

Because everything lives locally, deleting tokens.json removes the stored credentials completely. Access can also be revoked from TikTok's own settings (Security & permissions → Manage app permissions), which invalidates the token immediately.
