1. What this software is

MythFlash Uploader ("the Tool") is a personal command-line program written and operated by the creator of the MythFlash channel ("the Operator"). The Tool uploads video files from the Operator's own computer to the Operator's own TikTok account as drafts, using TikTok's official Content Posting API.

2. Who may use it

The Tool is for the Operator's sole use. It is not offered to the public, has no registration, and creates no accounts. It never acts on behalf of any TikTok user other than the single account that has explicitly authorized it through TikTok's OAuth consent screen.

3. What it does with TikTok

Once authorized, the Tool requests the video.upload permission and uses it for exactly one purpose: placing video files into the authorizing account's TikTok drafts. It does not publish content, does not read or modify existing posts, does not follow, comment, message, or interact with other accounts, and does not collect information about other TikTok users. Drafts become public posts only when a human reviews and publishes them inside the TikTok app.

6. Revoking access

Authorization can be withdrawn at any time from TikTok's settings (Security & permissions → Manage app permissions). Doing so immediately stops the Tool from being able to upload anything.
